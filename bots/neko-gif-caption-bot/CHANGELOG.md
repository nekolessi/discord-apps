# Changelog

## v1.0.0 - 2026-03-11
- Initial public self-hosted release.
- Added `/captiongif` command for GIF/APNG captioning.
- Added selectable fonts, outline levels, and optional background box.
- Added output format modes: auto, GIF, APNG.
- Added Docker and compose support for deployment.
- Added runtime limits and cooldown safeguards.
